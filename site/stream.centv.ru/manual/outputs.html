Выходы (Output)

Выход — это “куда отдавать” поток. В одном канале можно сделать несколько выходов.

Как добавить выход

  1. Откройте канал (Edit stream).
  2. В блоке OUTPUT LIST нажмите NEW OUTPUT или +.
  3. Выберите тип и укажите адрес.
  4. Нажмите OK, потом внизу Save.

Что выбрать в начале

  • UDP — самый простой вариант для локальной сети.
  • HTTP‑TS — удобно проверить в VLC по ссылке.
  • HLS — удобно для браузера и плееров, но это уже “плейлист + сегменты”.

Проверка проигрыванием

Ниже — примеры команд. Подставьте свой сервер и ID канала.

# HTTP-TS (пример)
ffplay -fflags nobuffer -flags low_delay -probesize 32 -analyzeduration 0 http://SERVER:9060/live/STREAM_ID

# HLS (пример)
ffplay http://SERVER:9060/hls/STREAM_ID/index.m3u8

# UDP (пример)
ffplay -fflags nobuffer -flags low_delay udp://239.0.0.1:1234
Если не играет — сначала убедитесь, что вход “живой” (есть битрейт).

Если выход не работает

  • Проверьте, что output включён (ON).
  • Если UDP multicast: клиент должен быть в той же сети и “видеть” multicast.
  • Если HTTP‑TS/HLS: проверьте, что порт открыт в firewall.
  • Посмотрите раздел “Если не работает”.
