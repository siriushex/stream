Входы (Input)

Вход — это адрес, откуда Stream Hub берёт поток. Обычно вы просто вставляете URL.

Как добавить вход

  1. Откройте канал (Edit stream).
  2. В блоке INPUT LIST нажмите NEW INPUT.
  3. Вставьте адрес источника.
  4. Нажмите OK, потом внизу Save.

Примеры адресов

Эти примеры — чтобы понять формат. Ваш адрес может быть другим.

# UDP multicast (пример)
udp://239.0.0.1:1234

# UDP с интерфейсом (когда важно, откуда принимать)
udp://enp5s0f1@239.0.0.1:1234

# HTTP (пример)
http://example.com:8000/live/channel
Если вы не уверены — начните с простого UDP/HTTP адреса.

Как понять, что вход “живой”

  • В карточке канала появляется битрейт (не 0).
  • Статус становится ONLINE.
  • В Analyze видно, что приходят пакеты.

Если вход не работает

  • Проверьте адрес. Ошибка в одном символе — и сигнала не будет.
  • Если UDP multicast: убедитесь, что сервер “видит” этот multicast (маршруты, интерфейс).
  • Если HTTP: попробуйте открыть URL в браузере/через curl.
  • Посмотрите раздел “Если не работает”.
