Stream Hub

Stream Hub принимает входной поток и отдаёт его дальше. Настройка — в веб‑панели.

  • Создаёте канал (Stream).
  • Добавляете вход (откуда брать).
  • Добавляете выход (куда отдавать).
  • Проверяете, что всё играет.
Начать за 10 минут Скачать бинарник Руководство
Перекодирование (Transcoding) — опционально. Если нужно, поставьте ffmpeg.

Установка одной командой

Вариант для Ubuntu/Debian. Для CentOS и macOS есть команды в “Быстром старте”. 

curl -fsSL https://stream.centv.ru/install.sh | sudo bash -s -- --mode binary --runtime-only
Поставит Stream Hub в /usr/local/bin/stream

Запуск

  • Запуск: stream -c /etc/stream/prod.json -p 9060
  • Web UI: http://SERVER:9060
  • Создайте канал (New Stream) и добавьте вход/выход.
