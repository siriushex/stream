# Первый канал (Stream)

## Шаги в UI

1. Откройте **Dashboard**.
2. Нажмите **New Stream**.
3. Заполните:
   - **Name** — как вам удобно.
   - **ID** — коротко, латиницей (например `news1`).
4. В **INPUT LIST** нажмите **NEW INPUT** и вставьте адрес входа.
5. В **OUTPUT LIST** нажмите **NEW OUTPUT** и добавьте выход.
6. Нажмите **Save**.

## Как понять, что всё работает

- В карточке канала появился битрейт (не 0).
- Статус стал ONLINE.
