# Быстрый старт

Эта инструкция — чтобы поднять Stream Hub и сделать первый рабочий канал.

## Коротко

- Устанавливаем.
- Запускаем на порту (пример: `9060`).
- Создаём Stream и проверяем, что “играет”.

## Что нужно заранее

- Linux‑сервер и доступ `sudo`.
- Свободный порт для Web UI.
- Адрес входного потока (например `udp://...` или `http://...`).
