# HLSSplitter

HLSSplitter — это отдельный сервис, который “режет” HLS и раздаёт его дальше.
В Stream Hub он управляется как “managed service”: вы настраиваете инстансы и ссылки в UI.

## Из чего состоит

- **Instance** — запущенный процесс HLSSplitter (порт, пути, логирование).
- **Link** — правило “входной URL → выходной URL”.
- **Allow rules** — кто может подключаться (IP‑правила).

## Быстрый старт

1. Откройте вкладку **HLSSplitter**.
2. Нажмите **NEW INSTANCE** и сохраните настройки.
3. Создайте **Link**:
   - Input URL (HTTP),
   - Output port/path (как выдавать наружу).
4. Проверьте ссылку в браузере/плеере.

!!! warning "Ограничение"
    Входы для HLSSplitter обычно принимаются только по HTTP (не HTTPS).

## Если не работает

- проверьте порт инстанса (занят/закрыт),
- посмотрите логи HLSSplitter,
- убедитесь, что input URL открывается с сервера.
