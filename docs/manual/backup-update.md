# Бэкап и обновление

## Перед любыми изменениями

Сделайте бэкап:

- конфигов (`/etc/stream/*.json`)
- env‑файлов (`/etc/stream/*.env`)
- data‑директории, если вы её используете (`--data-dir`)

## Обновление (если сервис systemd)

1. Скачайте новый бинарник.
2. Замените `/usr/local/bin/stream`.
3. Перезапустите сервис.

```bash
sudo systemctl restart stream@prod.service
```

!!! tip "Проверка"
    После рестарта откройте UI и проверьте 1–2 канала.
