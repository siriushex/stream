# Settings → Telegram alerts

Stream Hub умеет отправлять уведомления в Telegram.
Это удобно для дежурства: упал вход, сработал датчик качества, появился алерт.

## Что нужно

- Bot token (токен бота).
- Chat ID (куда отправлять сообщения).

## Что отправляется

Обычно в Telegram уходят события из **Alerts**:

- вход упал,
- сработал Quality detector,
- системные предупреждения (если включены).

## Дополнительно

В некоторых сборках доступны:

- плановый бэкап в Telegram,
- плановая сводка (summary) по инстансу.

## Проверка

1. Заполните token и chat id.
2. Нажмите **Test** (если кнопка есть).
3. Убедитесь, что сообщение пришло.
