# MPTS (Multi Program Stream)

MPTS — это поток, в котором “упаковано” несколько программ (каналов) в один транспортный поток.

Это используют, когда нужно:

- собрать несколько каналов в один UDP/RTP,
- отдать на оборудование, которое ждёт MPTS (например модуляторы).

## Как включить

1. Откройте канал (**Edit stream → General**).
2. Включите **Create Multi Program Stream**.
3. Настройте список сервисов (какие программы включать).
4. Добавьте выход (обычно UDP/RTP).
5. Нажмите **Save**.

## Как проверить

1. Откройте выходной UDP/RTP в анализаторе или плеере.
2. Убедитесь, что внутри несколько программ (PAT/PMT/SDT).

!!! tip "Если вы не уверены"
    Начните с одного сервиса. Потом добавляйте остальные.
