# Settings → Groups

Groups — это “группы каналов”.
Они удобны, когда вы:

- делаете плейлист и хотите группировать каналы (например “Новости / Спорт / Детские”),
- хотите быстрее фильтровать/находить каналы в UI.

## Как использовать

1. В Settings → Groups создайте группу (ID + Name).
2. В канале (Edit stream → Groups) выберите группу.
