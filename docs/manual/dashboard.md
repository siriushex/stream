# Dashboard

Dashboard — это главный экран Stream Hub. Здесь вы видите все каналы и их состояние.

## Что здесь можно сделать

- Создать новый канал (**New Stream**).
- Открыть настройки канала (окно **Edit stream**).
- Быстро понять, где проблема: вход, выход или транскод.
- Переключить вид списка: **Cards / Table / Compact**.
- Найти канал по имени или ID (поиск в верхней строке).

## Как быстро читать карточку канала

- **ONLINE** — поток идёт.
- **Bitrate** (например `1398 Kbit/s`) — если `0`, сигнала обычно нет.
- **Active input** — какой вход сейчас используется (если входов несколько).
- **Inputs: OK / DOWN / UNK** — короткая сводка по входам.

!!! tip "С чего начать при проблемах"
    Сначала смотрите вход: статус и битрейт. Потом — выход и firewall.

## Table view (таблица)

Таблица удобна, когда потоков много.
Обычно в ней видно:

- вход (адрес + аптайм + битрейт),
- выходы и клиентов,
- состояние транскода (если включён).

## Где смотреть клиентов

Если к вам подключаются по `http://SERVER:PORT/play/...` или `.../live/...`,
клиентов удобнее смотреть в разделе **Sessions**.
