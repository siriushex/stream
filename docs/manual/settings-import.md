# Settings → Import

Import нужен, чтобы быстро загрузить каналы в Stream Hub.

Поддерживаются форматы:

- JSON (экспорт/конфиг Stream Hub),
- M3U (плейлист).

## Режимы

- **Merge (add/update)** — добавить новые и обновить существующие.
- **Replace (wipe existing)** — заменить всё (осторожно).

!!! danger "Replace"
    Используйте Replace только если вы точно понимаете, что делаете. Он может удалить текущую конфигурацию.

## Как проверить

После импорта:

1. Откройте Dashboard.
2. Проверьте, что каналы появились.
3. Откройте 1–2 канала и убедитесь, что входы/выходы на месте.
