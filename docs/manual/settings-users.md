# Settings → Users

Users — это учётные записи для входа в Web UI и API.

## Что здесь обычно делают

- меняют пароль администратора,
- создают отдельные логины для инженеров,
- отключают лишние учётки.

## Сброс пароля админа

Если забыли пароль и есть доступ к серверу:

```bash
sudo /usr/local/bin/stream --reset-password
```

После этого логин/пароль снова `admin/admin`.

!!! warning "После сброса"
    Сразу зайдите в UI и поменяйте пароль на свой.
