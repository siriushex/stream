# EPG (экспорт)

EPG — это выгрузка телепрограммы (guide) для канала.

В Stream Hub настройка EPG обычно хранится **внутри канала** (вкладка EPG).
А интервал экспорта задаётся **в Settings → General** (если эта опция включена в UI).

## Как включить для канала

1. Откройте канал (**Edit stream**).
2. Перейдите на вкладку **EPG**.
3. Заполните:
   - **XMLTV channel ID** (например `channel.id`),
   - **Format** (XMLTV/JSON),
   - **Destination** (куда писать файл),
   - **Codepage** (если нужно).
4. Нажмите **Save**.

Если поля пустые — EPG для этого канала считается выключенным.

## Как проверить

1. Подождите один интервал экспорта.
2. Проверьте, что файл появился на диске (`Destination`).
3. Если файла нет — смотрите **Log**.
