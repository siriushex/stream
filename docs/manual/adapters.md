# Adapters (DVB)

Раздел **Adapters** нужен, если вы принимаете сигнал с DVB‑карт (DVB‑S/S2, DVB‑T/T2, DVB‑C).

Если у вас только IP‑входы (UDP/HTTP/HLS) — этот раздел можно не использовать.

## Что вы увидите

Обычно в списке адаптеров видно:

- ID адаптера,
- включён/выключён,
- краткий статус (busy/free, lock, SNR и т.д. — зависит от драйвера).

## Типовой сценарий

1. Откройте **Adapters**.
2. Нажмите **NEW ADAPTER**.
3. Выберите нужные параметры (адаптер/устройство/тип).
4. Нажмите **Save**.
5. В канале добавьте вход `dvb://...` (или выберите адаптер в UI).

## Полезные CLI команды

Список DVB адаптеров:

```bash
/usr/local/bin/stream --dvbls
```

Мониторинг сигнала (пример, параметры зависят от вашей сети):

```bash
/usr/local/bin/stream --femon dvb://...
```

!!! tip "Если /dev/dvb нет"
    Значит на сервере нет DVB устройств (или нет драйвера). Stream Hub для IP‑входов будет работать без DVB.
