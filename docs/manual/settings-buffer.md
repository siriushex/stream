# Settings → Buffer

Здесь настраивается **глобальный** сервер Buffer (HTTP‑TS buffer).

Если Buffer вам не нужен — оставьте выключенным.

## Основные поля

- **Enable HTTP TS buffer server** — включить/выключить Buffer.
- **Listen host / port** — где слушать сервер Buffer.
- **Source bind interface** — какой интерфейс использовать для чтения источников (если важно).
- **Max clients** — общий лимит клиентов.
- **Client read timeout** — таймаут чтения для клиентов (защита от “висящих” соединений).

## Как проверить

1. Откройте вкладку **Buffer** и создайте 1 ресурс.
2. Откройте ссылку ресурса в плеере.
3. Проверьте **Sessions** (клиент должен появиться) и логи.
