# Access (журнал доступа)

Access — это журнал “кто и что открывал”.
Он полезен, когда:

- кто‑то не может открыть ссылку,
- вы проверяете авторизацию,
- нужно быстро найти IP клиента.

## Что обычно видно

- время,
- IP,
- логин (если включена авторизация),
- какой stream / какой URL открывали,
- результат (OK/ошибка).

## Частые причины ошибок

- порт закрыт firewall’ом,
- неправильный Public URL,
- включена авторизация, но клиент без токена/логина,
- ссылка ведёт на другой сервер/порт.
