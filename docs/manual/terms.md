# Термины

Короткий словарь, чтобы было проще читать UI и логи.

- **Stream** — канал.
- **Input** — откуда берём поток.
- **Output** — куда отдаём поток.
- **ONLINE** — поток идёт.
- **OFFLINE** — поток не идёт (чаще всего проблема во входе).
- **Bitrate** — скорость потока. Если 0, обычно нет сигнала.
- **Transcoding** — перекодирование. Нужно, если хотите изменить качество/битрейт.
