# Входы (Input)

Вход — это адрес источника.

## Шаги

1. Откройте канал (Edit stream).
2. В **INPUT LIST** нажмите **NEW INPUT**.
3. Вставьте адрес.
4. Нажмите **OK**, потом **Save**.

## Примеры адресов

```text
udp://239.0.0.1:1234
udp://enp5s0f1@239.0.0.1:1234
http://example.com:8000/live/channel
```

!!! tip "Если не уверены"
    Начните с самого простого адреса. Если нужно выбрать интерфейс — добавьте `iface@`.

## Как понять, что вход живой

- В карточке появился битрейт.
- Статус стал ONLINE.

## Если вход не работает

- Проверьте адрес. Ошибка в одном символе — и сигнала не будет.
- Для multicast проверьте сеть и интерфейс.
