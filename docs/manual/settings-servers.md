# Settings → Servers

Servers — это список внешних серверов, которые Stream Hub может использовать в интеграциях.

Обычно сюда добавляют:

- адрес,
- порт,
- логин/пароль (если требуется),
- тип сервера (если есть выбор).

## Для чего нужно

- быстрый тест доступности/логина,
- удобное хранение параметров для нескольких площадок.

Если вы не используете внешние серверы — этот раздел можно пропустить.
